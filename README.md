# Adopta-ya
AdoptaYa es una aplicación web desarrollada con tecnologías frontend que promueve la adopción responsable de mascotas, permitiendo a los usuarios visualizar animales disponibles, conocer su información básica y simular una solicitud de adopción.

## Tecnologías utilizadas
 - HTML5
 - CSS3
 - Bootstrap 5
 - JavaScript
